# 🌸🌧️🍂❄️ 四季动画效果使用指南

## 功能特性

这是一个升级版的四季动画系统，比单纯的雪花效果更加丰富和智能！

### 🎯 四季效果

- **🌸 春季**：粉色花瓣飘落，带有旋转动画
- **🌧️ 夏季**：蓝色雨滴下落，模拟夏日雨景
- **🍂 秋季**：彩色落叶飞舞，多种暖色调
- **❄️ 冬季**：白色雪花飘落，经典冬日效果

### 🎮 智能控制

1. **自动模式**：根据当前月份自动切换季节
   - 3-5月：春季花瓣
   - 6-8月：夏季雨滴
   - 9-11月：秋季落叶
   - 12-2月：冬季雪花

2. **手动模式**：通过导航栏右侧的季节切换器手动选择

### 🎨 技术特性

- **Canvas 渲染**：使用 HTML5 Canvas 实现高性能动画
- **响应式设计**：自动适配不同屏幕尺寸
- **性能优化**：
  - 粒子数量自动调节
  - 页面不可见时暂停动画
  - 支持用户动画偏好设置
- **无障碍支持**：遵循 `prefers-reduced-motion` 设置

### 🔧 使用方法

1. **查看效果**：访问主页即可看到当前季节的动画效果
2. **切换季节**：点击导航栏右侧的季节切换器
3. **自动模式**：选择"🔄 自动切换"让系统根据当前月份自动选择
4. **手动模式**：选择具体季节来固定显示某种效果

### 📱 兼容性

- ✅ 现代浏览器（Chrome、Firefox、Safari、Edge）
- ✅ 移动设备响应式适配
- ✅ 高分辨率屏幕支持
- ✅ 暗色/亮色主题兼容

### ⚙️ 自定义配置

动画效果会自动保存用户的选择到 localStorage，下次访问时会记住你的偏好设置。
